# System - Monolith (C#)

## Instructions

Check that you have Powershell 7

```shell
$PSVersionTable.PSVersion
```

Ensure you have .NET 8:

```shell
dotnet --version
```

To build the project:

```shell
dotnet build
```

To run:

```shell
dotnet run
```

App should now be running on:
http://localhost:8080/
